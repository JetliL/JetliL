One of the most important elements that are needed for mobile applications, or any life applications are the principles of software engineering. This article presents a development of process of based on transformation UML models to Mobile Maude Language. UML stands for Unified Modeling Language in the field of Software Engineering. It is standardized as a visual modeling language and is used for specifying, visualizing, constructing, and documentation the artifacts of the software system. Example's such as class diagrams, component diagram, deployment diagram, object diagram, package diagram, and ect. The UML diagrams will produce located configurations which will be supporting generated rewriting theories that get executed for simulation. Verified models of mobile applications will be used to generate Android application to be deployed on mobile devices.
